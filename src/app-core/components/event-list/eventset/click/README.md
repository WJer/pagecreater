# 点击事件

layer对象：
{
  ...,
  events: [
    {
      id: '', // 随机id
      eventId: 'h5ds_event_click', // 事件标识
      eventParam: {
        value: [], // 参数. 是一个 string，多个参数使用 , 分割
        eventName: '', // 事件名称 h5ds_show_modal, h5ds_show_hide_layer, h5ds_custom_event_name
      },
      name: '点击事件' // 事件的名称描述
    }
  ]
}
